Sybase와 MS SQL Server

MSSQL Server의 전신이 Sybase에서 개발한 DBMS라는 신기한(?) 말을 들었다. 독자적으로 개발한 게 아닌가? MS가 Sybase를 인수라도 한 것일까? 좀 더 자세히 알아보기로 했다. 관련 키워드로 검색했더니 ‘Microsoft가 인수를 통해 확보한 인기 제품들’ 이란 IT World의 기사를 찾았다. 그런데 여기서 소개하는 SQL Server 페이지는 조금 다른 이야기를 하고 있었다. 인수라고 말하기도 뭣하고, 그렇다고 MS가 개발한 것도 아니라는 것이다. 사실은, 다음 순서대로 사건(?)이 발생했다. 1987년, Unix 플랫폼에서 구동되는 Sybase에서 개발한 DBMS가 있었다....
