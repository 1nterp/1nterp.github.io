Confluence Code Macro 빠르게 넣기

(애니메이션 예제는 SQL이다.) 콜론을 찍으면서 자동 완성이 사라지는데 당황하지 말고 전부 입력하면 된다. 원래는 {code:language=<language>} 같이 입력하도록 Confluence 매뉴얼에서 안내하고 있는데, 매크로의 맨 처음 파라메터는 그 attribute를 생략해도 인식되기 때문에 이게 가능하다....
